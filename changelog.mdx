## Recent Releases

Stay current with Splan's evolution. You access the latest version through your dashboard at `https://dashboard.example.com`. Review these updates to leverage new capabilities in your real estate sales and marketing automation.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added AI-powered lead scoring that analyzes prospect behavior and property interests to prioritize high-conversion leads.
- Introduced automated email campaign personalization using property data integration.

## Improvements

- Enhanced dashboard performance with real-time analytics for sales pipelines.
- Optimized CRM sync to handle `>10,000` leads per hour without delays.

## Bug Fixes

- Resolved issue where funnel reports showed incorrect conversion rates.
- Fixed webhook delivery failures during peak usage.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Launched property listing importer from major MLS systems.
- Added multi-channel campaign builder for SMS, email, and push notifications.

## Bug Fixes

- Corrected authentication token refresh logic to prevent session timeouts.
- Patched vulnerability in webhook endpoint validation.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core sales automation engine with lead nurturing workflows.
- Integrated marketing analytics dashboard for ROI tracking.

## Breaking Changes

- Updated API v2 endpoints; migrate from v1 by updating `{baseUrl}` to `https://api.example.com/v2`.

</Update>
