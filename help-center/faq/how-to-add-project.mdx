## Быстрый старт

Для добавления нового проекта выполните несколько простых шагов.

1. Перейдите в раздел "Проекты" на панели управления.
2. Нажмите кнопку "Добавить проект".
3. Введите название, адрес и описание проекта.
4. Загрузите презентацию или планировки (по желанию).
5. Сохраните проект.

<Callout kind="info">
Проект появится в списке сразу после сохранения.
</Callout>
